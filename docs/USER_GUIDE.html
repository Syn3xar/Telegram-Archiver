Telegram Admin GUI User Guide

Step-by-step instructions for authorized group administrators
Built by Syn3xar

Use this app only in groups where you have permission to administer content.

1. Open the App

  1. Open the folder where the app was provided.
  2. Double-click the app.
  3. Wait for the Telegram Admin Tool window to appear.

2. Fill In the Fields

  • Admin user ID: the Telegram user ID for the admin account.
  • Group ID: the group where the admin account has permission to operate.
  • Target user ID: the account you want to search, review, or verify before deletion.
  • Phone: the Telegram phone number for the admin account in international format.
  • API ID / API hash: the Telegram API credentials assigned to the app.
  • Two-step verification password: the account password if two-step verification is enabled.
  • Batch size: the number of messages read per page. Reduce this value if Telegram disconnects often.
  • Total streams: maximum simultaneous media downloads. The recommended value is 4.
  • Small streams: files up to 20 MB. The recommended value is 4.
  • Large streams: files over 20 MB or with unknown size. The recommended value is 2.
  • Output folder: where reports, archives, and downloaded media should be saved.

3. First Login

  1. Click an action such as Search Target Messages.
  2. If Telegram asks for a login code, enter the code received by the admin account.
  3. If Telegram asks for two-step verification, enter the account password.
  4. After login, a local session is saved so future launches usually do not need another code.

4. Search Target Messages

  1. Confirm the Target user ID.
  2. Click Search Target Messages.
  3. Found messages appear in the detected messages table.
  4. Use the message ID column for previewing or deleting.

The app also saves a search report in the selected output folder.

5. Preview a Message

  1. Select a row in the message table, or type a message ID.
  2. Click Preview Message.
  3. Read the sender, date, media type, and text preview in the status screen.

6. Delete a Message

  1. Search for the target account's messages first.
  2. Select the message row.
  3. Click Preview Message and confirm it is correct.
  4. Click Delete Message.
  5. Confirm the warning.
  6. Type exactly DELETE.
  7. The app verifies the sender, deletes, then asks Telegram again whether the message is gone.

If the sender does not match the target user, the app refuses to delete by default.

7. Archive Text Chat

  1. Click Archive Text Chat.
  2. Wait for the scan.
  3. Open the saved report from the selected output folder.

The archive includes text messages and message metadata.

8. Download Media

  1. Choose whether to check Only target media.
  2. Click Download Media.
  3. Wait while visible group history is scanned and the remaining known size is reported.
  4. Watch the overall progress bar for processed quantity, remaining quantity, and percentage.
  5. Watch the labelled file-level worker progress in the status screen.

If Only target media is checked, the app downloads media only from the target account. Otherwise, it downloads media from all visible messages in the group.

Media is saved in a stable folder. Complete files are skipped on later runs. Temporary server errors, flood waits, and cancelled transfers are retried per worker; an item that still fails remains pending without stopping the other downloads.

9. Stop a Long Task

Click Stop Current Task. During media downloads, active transfers are cancelled cleanly and incomplete files are removed.

10. Understanding Output

The app can create searchable message reports, readable chat archives, downloaded media folders, and media manifests for review.

11. Troubleshooting

  • Login code prompt: enter the code received by the admin account.
  • Two-step verification prompt: enter the account password.
  • Wrong logged-in account: close the app, remove the saved local session if needed, and log in with the correct admin account.
  • No messages found: check the target user ID, group ID, and visible group history.
  • Telegram disconnects: reduce the batch size and run the action again.
  • Delete fails: the account may lack permission, the message may already be deleted, or Telegram may not allow deleting that message.

12. Recommended Safe Workflow

  1. Enter the admin user ID, group ID, and target user ID.
  2. Click Search Target Messages.
  3. Select the message.
  4. Click Preview Message.
  5. Confirm it is the correct message.
  6. Click Delete Message.
  7. Type DELETE.
  8. Check the confirmation line in the status screen.
